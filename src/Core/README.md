`Core` — это расширяемая основа проекта, организованная по принципу "Вертикальных срезов" (Vertical Slices).

### Структура:
- **Foundation/**: Базовые механизмы движка (Engine, Unit, Systems, Components).
- **Modules/**: Доменно-ориентированные модули игровых механик:
  - **Spatial/**: Позиционирование и перемещение.
  - **Combat/**: Боевая система и разрешение атак.
  - **Vitals/**: Здоровье и жизненный цикл юнитов.
  - **Stats/**: Характеристики персонажей (Strength, Agility, Speed).
  - **Rng/**: Система генерации случайных чисел.
